# Automobile Sales Dashboard: Insights from Macroeconomic Trends

This project explores automobile sales in relation to macroeconomic trends, including recessions and unemployment rate. It includes an **analysis notebook** and an **interactive Dash application** (code only).

---

## Project Contents

### 1. Analysis Notebook
- `Automobile_Sales_Analysis.ipynb` → Jupyter notebook explaining the project and displaying key graphs.

### 2. Dash Application
- `app_dash.py` → Python code for the interactive Dash dashboard.  

---

## Technologies Used

- Python 3.8  
- Dash
- Plotly
